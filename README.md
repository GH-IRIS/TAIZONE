## 🛠 Tech Stack
HTML5 · CSS3 · Clerk (Google authentication) · AI-assisted interactivity (Antigravity Code Assistant) · GitHub Pages

## 🎨 Design references
Inspired by premium minimalist e-commerce layouts from Godly.website and Aceternity UI.

## 🤖 How I used AI
I built the page layouts, structures, and CSS styles; AI generated the product list catalog database, designed the reactive cart arithmetic calculations, and structured the Clerk auth with mock auth fallback interfaces.

## 📚 What I learned
I mastered single-page application hash-based routing in vanilla JavaScript, built a state-driven reactive cart that persists dynamically using browser local storage, and structured security barriers to protect the checkout views.
